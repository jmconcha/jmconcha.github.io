Featured Project

Blog

A social media web application, powered by React and Firebase. User can post, like, comment and delete a blog.

I used React for the user interface, Redux to handle the app state, React Router to handle app routes and to have a single page functionality, and Redux Thunk for easier management of Redux state and asynchronous calls. Some of the components are from Material-UI for beautiful and responsive UI components. Firebase authentication for login and signup functionality. Firebase storage for storing user profile pictures. Firebase firestore for user details, blog posts, likes, comments, and notifications.

I had so much fun building this project. It gives me a lot of challenges. I learned a lot about the technologies that I used, cool stuff!
